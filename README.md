## Description

[SuperNest](https://github.com/nestjs/nest) is a Blockchain indexer for the LGCY Blockchain. It parses the blocks starting by 0 and stores all data in a MongoDB.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
